HTML Elements

<!DOCTYPE>: Specifies the HTML version and document type.

<html>: The root element of an HTML document.

<head>: Contains meta-information about the document.

<title>: Sets the title of the document, displayed in the browser's title bar.

<body>: Contains the content of the document.

<header>: Represents introductory content at the beginning of a section.

<nav>: Defines a set of navigation links.

<main>: Represents the main content of the document.

<section>: Defines a section of the document.

<article>: Represents an independent piece of content.

<aside>: Contains content that is tangentially related to the main content.

<footer>: Contains information about the document or section.

<p>: Represents a paragraph of text.

<h1> to <h6>: Heading elements ranging from the most important to the least important.

<a>: Creates hyperlinks.

<img>: Embeds images into the document.

<ul>: Creates an unordered list.

<ol>: Creates an ordered list.

<li>: Represents a list item within a list.

<table>: Defines a table for organizing data.

<tr>: Defines a row within a table.

<td> and <th>: Define data cells and header cells within a table, respectively.
